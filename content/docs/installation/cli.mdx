docpler CLI는 독립 실행형 네이티브 바이너리입니다. Python이나 별도 런타임이 필요 없습니다.

<Tabs items={['macOS', 'Linux']}>
  <Tab value="macOS">
```bash
brew install docpler/tap/docpler
```
  </Tab>
  <Tab value="Linux">
```bash
curl -fsSL https://github.com/docpler/docpler-cli/releases/latest/download/docpler-x86_64-unknown-linux-gnu -o /usr/local/bin/docpler
chmod +x /usr/local/bin/docpler
```
  </Tab>
</Tabs>

> **Windows 사용자:** CLI 바이너리 대신 [Python 패키지](/docs/installation/python)로 설치해 주세요.
> ```bash
> pip install docpler
> ```

## 직접 다운로드

[GitHub Releases](https://github.com/docpler/docpler-cli/releases)에서 플랫폼에 맞는 바이너리를 다운로드할 수 있습니다.

| 플랫폼 | 바이너리 |
|--------|---------|
| macOS ARM | `docpler-aarch64-apple-darwin` |
| macOS Intel | `docpler-x86_64-apple-darwin` |
| Linux x86_64 | `docpler-x86_64-unknown-linux-gnu` |

## 설치 확인

```bash
docpler --version
```
