## 기본 표 (Free)

HWP 5.0의 표 구조를 파싱하여 Markdown 파이프 테이블로 변환합니다.
행, 열, 셀 텍스트가 정확하게 보존됩니다.

```markdown
| 이름   | 나이 | 직업     |
|--------|------|----------|
| 홍길동 | 30   | 개발자   |
| 김철수 | 25   | 디자이너 |
```

## 중첩표 (Pro)

<Callout>중첩표는 Pro 기능입니다.</Callout>

표 안에 표가 포함된 중첩 구조를 최대 3단계까지 감지합니다.
Pro에서는 HTML `<table>`로 렌더링하여 구조를 보존합니다.

```html
<table>
<tr><th>이름</th><th>성적 상세</th></tr>
<tr><td>홍길동</td><td>
  <table>
  <tr><td>수학</td><td>95</td></tr>
  <tr><td>영어</td><td>88</td></tr>
  </table>
</td></tr>
</table>
```

Free에서도 중첩표 셀의 텍스트는 평문으로 합쳐서 출력됩니다 (`수학 95 영어 88`).

## Typst / LaTeX

표는 각 출력 형식에 맞게 변환됩니다:

- **Typst** — `#table(columns: 3, ...)`
- **LaTeX** — `\begin{tabular}` + booktabs (`\toprule`, `\midrule`, `\bottomrule`)
