## 이미지 감지 (Free)

Free에서도 이미지 위치와 캡션을 감지합니다. 플레이스홀더로 출력됩니다.

```
이미지가 포함된 HWP 파일이에요.

[이미지: 그림 사진: Unsplash의 Vladislav Nahorny]
```

## 이미지 추출 (Pro)

<Callout>이미지 추출은 Pro 기능입니다.</Callout>

Pro에서는 HWP의 CFB 컨테이너에서 실제 이미지 바이너리(PNG, JPEG, GIF, BMP)를
추출하여 `assets/` 디렉토리에 저장합니다.

```bash
docpler convert document.hwp -o output.md

# 결과:
# output.md           ← ![캡션](assets/image001.png)
# assets/image001.png ← 실제 이미지
```

## ZIP 번들

`--format zip`으로 출력하면 Markdown과 이미지가 하나의 아카이브로 묶입니다.

```
document.zip
├── index.md
├── manifest.json
└── assets/
    └── image001.png
```

## Typst / LaTeX

이미지는 각 형식에 맞게 변환됩니다:

**Typst:**
```typ
#figure(
  image("assets/image001.png"),
  caption: [캡션 텍스트],
)
```

**LaTeX:**
```latex
\begin{figure}[htbp]
  \centering
  \includegraphics[width=0.8\textwidth]{assets/image001.png}
  \caption{캡션 텍스트}
\end{figure}
```
