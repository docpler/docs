## Free

설치 즉시 사용 가능. 라이선스 키 불필요.

```bash
pip install docpler
docpler convert document.hwp
```

HWP → 기본 Markdown 변환. 문단, 표, 수식, 글상자를 텍스트로 추출합니다.

## Pro

정밀 변환과 다양한 출력 형식이 필요한 개인 및 팀.

```bash
export DOCPLER_LICENSE_KEY="<your-jwt>"
docpler convert document.hwp -o output.md
# → 정밀 Markdown + 이미지 추출 자동 적용
```

- 라이선스 키 설정 시 `convert`가 자동으로 정밀 Markdown 적용
- `--format typst`, `--format latex` 등 추가 출력 형식
- 이미지 자동 추출 (`assets/` 디렉토리)
