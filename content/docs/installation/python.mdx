## 요구사항

- Python 3.9+

## PyPI에서 설치

```bash
pip install docpler
```

## 지원 플랫폼

| 플랫폼 | 아키텍처 |
|--------|---------|
| macOS | ARM (Apple Silicon) |
| macOS | x86_64 (Intel) |
| Linux | x86_64 |
| Windows | x86_64 |

## 설치 확인

```python
from docpler.hwp import convert
print(convert("document.hwp"))
```
